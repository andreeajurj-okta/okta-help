Setup Access Gateway Using an OVA Image

Overview

The purpose of this guide is to walk through the process of installing and configuring Access Gateway instance using an OVA image.

Prerequisites

  • Access Gateway must be resolvable via DNS and is required to have a hostname assigned in DNS.

  • Static IP Address - Access Gateway requires a static IP address and does not leverage DHCP.

If end users will be originating from the internet, the Access Gateway solution must be publicly resolvable. It is recommended that split DNS be leveraged so that internal users connect to an internal IP address and external users connect to a public IP address.
