Downloads

Okta provides a Downloads page where you can easily access the latest browser plugins, .msi files (Windows Installer), and agent downloads. It also displays the latest software versions, with access to a version history for all downloads.

On the Downloads page, click Download for the plugin or administrator component you require.

Note that not all components have a corresponding Download button; for example, the Chrome plugin. For such components, we provide a link to the appropriate download store, such as the Google Play Store, etc.
