Okta Mobility Management Features and Support

OMM Version Support

iOS

Okta currently supports iOS versions 8.x, and 9.x.

Note: Support for iOS versions 7.x ended on March 1, 2016.

Android

Okta supports Android versions 4.0.4 and higher.

Support Policy

The Okta's iOS Support Policy is to support current and previous versions of iOS.

We target end of support for the oldest supported iOS version two months after a new version is released. For example, if iOS 10 is released on September 15, 2016, we will end support for iOS 8 two months later on November 15, 2016.

After the support period ends for a given iOS version, existing installations on the given version remain unaffected for the short-term. However, we encourage customers to move their end-users off of older version as future updates to Okta Mobile and Okta Verify may break compatibility with the app running on unsupported iOS versions.

Additionally, when support for an iOS version ends, end users will no longer be able to install Okta Mobile or Okta Verify on the affected iOS version.
