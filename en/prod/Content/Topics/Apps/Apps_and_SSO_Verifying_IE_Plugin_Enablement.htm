Verifying IE Plugin Enablement

You can perform either of the following procedures to verify that your Okta Internet Explorer plugin is enabled. If the first procedure does not enable your plugin, try the second procedure.

Verify that ActiveX Filtering is Disabled

  1. In Internet Explorer, select the Tools menu. If ActiveX Filtering is checked, clear it.

  2. Bookmark this page, then close Internet Explorer.
  3. Restart the browser, then return to this page and click here to verify whether your plugin has been successfully enabled.

Manually Enable Okta Add-ons

  1. Go to Tools > Internet Options.
  2. Click the Advanced tab.
  3. In the Browsing section, make sure Enable third-party browser extensions is selected. If it isn’t, select the option, click Ok, restart your computer, and then proceed to step 4.
  4. Go to Tools > Manage add-ons.
  5. In the Manage Add-ons window, under Show, select All add-ons from the drop-down menu.
  6. In the right panel, under Okta, Inc, select Okta toolbar and then click Enable.

  7. Select OktaBHO Class and then click Enable.

  8. Close the Manage Add-ons window.
  9. Bookmark this page, then close Internet Explorer.
  10. Restart the browser, then return to this page and click here to verify that your plugin has been enabled successfully.
