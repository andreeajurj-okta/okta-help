Admins can enable and disable this feature in Self Service Applications settings as follows:

  1. Go to Applications > Self Service.

  2. Click Edit.

  3. Select the following:

    1. Enable self service application assignment.

    2. Select either Show organization and user managed apps only or Show user managed apps only.

      The feature is disabled if you select Show organization managed apps only.

  4. Click Save.
