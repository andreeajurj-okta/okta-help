System Log

The system log panel on the main reports page displays a list of events. Clicking any of the links displays the system log for those events. Selecting System Log at the top of the list displays all logged events for the last 30 days with default parameters.

After the reports displays, you can change any of the parameters, including the date range, and run the report again.

Accessing the New System Log

The system log panel on the main reports page displays a list of events. Clicking any of the links displays the system log for those events. Selecting System Log at the top of the list displays all logged events for the last 30 days with default parameters.

You can also navigate to Reports > System Log. The system log displays all events. This is equivalent to selecting System Log at the top of the list.

System Log Graphs

The system log displays the following bar graphs about your chosen events:

  • Count of events over time
  • Count of events per target
  • Count of events per actor
  • Count of events per event type

For more information about a data point, hover over any of the bars in the graphs.

Events Table

This table lists all events and includes the following information: Time, Event Type, Message, Outcome, Actor, and Targets.

  • You can click the right arrow on any row to display more data about that event.
  • You can click on any Event Type or Actor in the table, to filter by that item.
  • You can download the entire table by clicking the Download CSV file link.

Customizing Your System Log

You can customize your System Log with the following specifications:

Date Range: Specify a start and end time range to filter the events displayed. The default is the preceding 7 days.

  • Time Zone: Use the dropdown menu to select a time zone in which the system log events are displayed.

  • Search: There are two types of search:

    • Basic: Enter a string to search all events within the specified time range. To do a basic search, type your search string, then click enter, or the search icon.

    • Advanced: Advanced search allows you to do a structured search based on the various fields of the events.

      To perform an advanced search: Click Advanced Search, then enter your selection criteria then click Apply Filter.
