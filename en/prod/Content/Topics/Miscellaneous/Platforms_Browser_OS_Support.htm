Windows browsers

For components accessed from the Web, such as the Okta Browser Plugin and Okta Sign-In Widget, Okta supports Windows Desktop editions 7, 8, and 10, and the following versions of Internet Explorer and Edge:

Internet Explorer for Windows

 

IE 9 IE 10 IE 11
Support ended July 15, 2016, including support for all versions of Compatibility View that represent Internet Explorer 9. Okta Support no longer investigates issues related to Internet Explorer 9. Support ended April 30, 2019, including support for all versions of Compatibility View that represent Internet Explorer 10. Okta Support no longer investigates issues related to Internet Explorer 10. Supported for the desktop mode of Windows 8 but not metro mode.
