Topic that is not context-sensitive

This topic is not context-sensitive. If you do not put it in the Table of Contents, users will not see it.

If you have a topic in progress, you can add it to the project and leave it out of the Table of Contents. You can then build and distribute a version of the project that will not show the unfinished topic. This let you create several topics to match a plan without worrying that users might see a empty of unfinished topic.
