Okta Mobile 5.2.0 for iOS

Okta Mobile 5.2.0 for iOS began deployment on December 20, 2016 and is available from the Apple AppAn abbreviation of application. Essentially, it is a web-based site used to perform any number of specific tasks, and requires authentication from end users by signing in. Store.

This release includes general performance improvements and bug fixes, including the following:

New Features

  • Support for iPad Pro screen resolution.
  • Enforced security updates. From version 5.2.0 and after, Okta will strongly encourage security sensitive updates. In some cases, including security vulnerabilities or crash threats, apps will be locked if an upgrade does not occur within a certain span of time.
  • Managed app configuration. This feature allows Okta Mobile to accept configurations sent from 3rd-party MDM providers. For details, see Preconfigure Site Name and Username with Managed Configurations.

Bug Fixes

  • OKTA-105564 - Okta Mobile failed to launch the correct app from Google Apps when launching as a native app.
  • OKTA-100824 - Launching a bookmark app from Okta Mobile fails for the iOS device, but works as expected for Android.

Announcement

This release marks the end of support for iOS 8. The last version to support iOS 8 was Okta Mobile 5.0.0.

Latest Okta Verify Releases

Okta Verify 2.8.0 for Android

Okta Verify 2.8.0 for Android began deployment on November 2, 2016, and is available from the Google Play Store.

Okta Verify now honors the language set on your device.

Bug Fixes

  • OKTA-96682 - The Okta Verify app crashed after the splash screen appeared.
  • OKTA-78777 - Okta Verify with Push failed to allow the device to communicate with the server.
Okta Verify 3.0.0 for iOS

Okta Verify 3.0.0 for iOS was released on November 1, 2016 and is now available from the Apple App Store.

This release includes the following:

  • A cleaner end user experience with a streamlined TouchID design.
  • Enhanced security, allowing view of the one-time passcode (OTP) only after a successful TouchID or PIN authentication.
  • Ability to authenticate multiple accounts and organizations, including accounts outside of Okta. Screenshot

  • Convenient, in-app ability to enable camera access for Okta Verify. Screenshot
