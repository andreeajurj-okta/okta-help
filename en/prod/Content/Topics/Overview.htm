Getting Started

This is the default topic in the project. This is a good place for general information and onboarding information.

Information about using the help system is included here, but you can move it to another topics, if desired.

 

Using this help system

The Help system uses the following icons shown below and described in the following table:

 

Help System Functions
Function Description
Hide/Show Navigation Shows or hides the pane to the left of the topic. This pane usually contains either the Table of Contents or the results of a search.
Home Returns to the Getting Started topic.
Print Prints the current topic.
Back Moves back one topic.
Search Search the Help topics for text you enter.
Table of Contents Shows the Table of Contents in the navigation pane.

 

Search Help

To search Help, clickeith the Search icon or the Search Bar below the table of contents, enter text in the search box, and click Search. The navigation pane contains the search results sorted by relevance. Click any result to display that topic. To view the results alphabetically, click the Title column header in the pane. To see the results in order of relevance again, click the Rank column header in the pane.

To start another search, enter a different search term in the box at the top of the help window and click Search.
