Administrators

Administrators (or Admins) are Okta users with permission to access the Okta administration application. You can grant Admins access to all sections of the application, or limit their access to only certain apps.

To add administrators, perform the following steps:

  1. Click Add Administrator.
  2. Enter the Administrator name.
  3. Define the administrator permissions by choosing an administrator type. For a full description of each admin type, click View permissions info.
  4. Once the fields are complete, click Save.
