QRコードをスキャンせずにOkta Verifyをセットアップする

初めてデバイスにOkta Verifyをセットアップするには 、ストアからアプリをダウンロードし、デバイスにイ ンストールします。 次にコンピューターのOktaにサインインして、手順に 従いデバイスを登録します。 デバイスでQRコードをスキャンできない場合、次の手 順に従ってください。

はじめに

  • デバイスによってOkta Verifyがサポートされていることを確認します。 「OktaによるブラウザーとOSのサポートポリシー」をご覧ください。
  • デバイスでQRコードのスキャンが有効になっていることを確認します。 詳細については、「Okta Verifyをインストールする」をご覧ください。 QRQRコードをスキャンしてOkta VerifyをセットアップするコードをスキャンしてOkta Verifyをセットアップする」
  • Okta Verifyのセットアップを開始する前に、[Oktaサイインイン]ページへのリンクおよびユーザー名を検索します。
    管理者からのOkta Welcome/Activation (Oktaに関する案内／有効化)の電子メールには、パスワードの作成とアカウントの有効化が可能になる、ユーザー名とリンクが含まれています。 案内の電子メールを受信しなかった場合、管理者にご連絡ください。
    • Oktaサインインページ：案内の電子メールの[Activate Okta Account (Oktaアカウントを有効化する)]ボタンの下にあります。
    • ユーザー名： この認証情報は管理者によって作成されます。 通常はユーザーの電子メールです。

この手順を開始する

  1. タスク1：デバイスにOkta Verifyをインストールする
  2. タスク2：コンピューターでOktaにサインインする
  3. タスク3：Okta Verifyにデバイスを登録する

タスク1：デバイスにOkta Verifyをインストールする

ストアからアプリをダウンロードして、インストールします。

  • iOS：Apple App Store - Okta Verify
  • Android： Google Play Store - Okta Verify

タスク2：コンピューターでOktaにサインインする

  1. コンピューターでウェブブラウザーを開き、OktaのサインインURLを入力します。「https://組織名.okta.com」 と入力し、「組織名」 は会社名または組織名を示します。
  2. ユーザー名（電子メールアドレス）とパスワードを入力します。
  3. [Sign In (サインイン)]をクリックします。

    Okta Verifyが組織によって義務付けられている場合、[Set up multifactor authentication (多要素認証をセットアップする)]ページが開きます。

    重要：セキュリティーキー（U2F）のオプションが表示される場合、管理者によってそれが追加要素として構成されています。 詳細については、管理者にご連絡ください。

  4. [Setup (セットアップ)]をクリックします。
  5. デバイスの種類を選択し、[Next (次へ)]をクリックします。
  6. [Scan barcode (バーコードをスキャンする)]ページで、QRコード下の[Can't scan? (スキャンできませんか？)]リンクをクリックします。

    QRコードをスキャンする代わりに、[Okta Verify (Okta Verifyをセットアップする)]画面にデバイスで使用する秘密鍵が表示されます。 この画面には2つのバージョンがあります。

    重要：下記の[Okta Verifyにデバイスを登録する]タスクの手順6に達するまで、[Setup Okta Verify (Okta Verifyをセットアップする)]画面の[Next (次へ)]をクリックしないでください。

  7. ブラウザーにどのバージョンの画面が表示されるかによって、次を実行します。
    • 画面にドロップダウンメニューがある場合、最適なオプションを選択し、画面の手順に従います。
      • Send activation link via SMS（SMSで有効化リンクを送信する）
      • Send activation link via SMS（SMSで有効化リンクを送信する）
      • Set up manually without Push Authentication（プッシュ認証せずに手動でセットアップする）。 このオプションを選択する場合、秘密鍵を書き留めておきます。 後の手順でデバイスのOkta Verifyアプリでこの秘密鍵を入力します。
    • 画面にドロップダウンメニューがない場合、秘密鍵を書き留めておきます。 後の手順でデバイスのOkta Verifyアプリで秘密鍵を入力する必要があります。

      重要：下記のタスクで手順5に達するまで、[Setup Okta Verify (Okta Verifyをセットアップする)]画面で[Next (次へ)]をクリックしないでください。

タスク3：Okta Verifyにデバイスを登録する

注意：持つことができるOkta Verifyの アカウントです。Okta VerifyアカウントはOkta Verifyの登録と呼ばれることがあります。 . Okta Verifyには1つのデバイスのみ登録できます。

  1. デバイスでOkta Verifyを開きます。
  2. アカウントを追加します。
    • iOS[Add Account (アカウントを追加する)]をタップします。
    • Androidプラス記号（+）をタップします。
  3. [Scan barcode (バーコードをスキャンする)]ページで、[No Barcode? (バーコードがありませか？)]をタップします。
    .
  4. [Add account (アカウントを追加する)]画面で、認証情報を入力し、[Add Account (アカウントを追加する)]をタップします。
    • Username（ユーザー名）：案内の電子メールで受け取ったユーザー名を入力します。
    • Secret key（秘密鍵）：コンピューターに表示されている鍵を入力します。 「タスク2：コンピューターでOktaにサインインする」の手順6をご覧ください。
  5. コンピューターのブラウザーの[Setup Okta Verify (Okta Verifyをセットアップする)]ページで[Next (次へ)]をクリックします。
  6. ブラウザーの[Enter code displayed from the application (アプリから表示されたコードを入力する)]ページで、デバイスのOkta Verifyでアカウントに表示される数字を入力します。
  7. [Verify (確認する)]をクリックします。

    8. これでOkta Verifyがデバイスにセットアップされました。Oktaで管理されている組織のアプリにサインインする際の認証に、これを使用できます。
