Preconfigure Site Name and Username with Managed Configurations

To improve the end-user experience, we have added support for managed configurations to allow 3rd party EMM providers to preconfigure the site name (domainA domain is an attribute of an Okta organization. Okta uses a fully-qualified domain name, meaning it always includes the top-level domain (.com, .eu, etc.), but does not include the protocol (https).) and username for the Okta Mobile apps for Android† and iOS.

†This feature works on Android devices managed by Android for Work.

Restriction elements

Key Value (example)
siteName acme
username jane.doe@acme.com

For more information, see Setting Up Managed Configurations (Android) or Managed Applications (Apple).
