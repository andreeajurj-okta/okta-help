SSO IWA Web App Version History

This page displays current and past versions of the Okta Integrated Windows Authentication (IWA) Web Application. This page is updated whenever a new version of the application is released (Generally Availability (GA) and/or Early Access (EA)). It was last updated March 29, 2017 (release 2017.13).
