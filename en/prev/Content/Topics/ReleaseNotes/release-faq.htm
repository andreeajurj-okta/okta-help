4. What's the naming convention for monthly releases and weekly updates?

    Monthly releases and weekly updates use the following convention based on their date of deployment:

    <year>.<month>.<release-type>

    For example:

    • 2019.04.0 — Monthly release for April 2019 (.04 = month number, .0 = monthly release)
    • 2019.04.2 — Weekly update for April 2019 (.04 = month number, .2 = second weekly update after monthly release)

 

 

5. What's the difference between Preview and Production releases?

    • Preview release: Features, enhancements, fixes, and integrations are deployed to preview orgs only.
    • Production release: Features, enhancements, fixes, and integrations are deployed to production orgs.

     

    Note: Most features and enhancements that are introduced in a preview release are typically made available in the production build of the next monthly release.

 

 

6. What are Early Access features and Generally Available features?

    Early Access (EA):  Opt-in features made available by request to Okta Support.

    Super Admins can enable or disable some EA features via the admin console under Settings > Features.

     

    Generally Available (GA): Features made available to all orgs automatically based on customer SKU.

 

 

7. How do I find release notes for a specific release?

    Release notes are published and maintained on help.okta.com.

    All release types are located under Release Notes in the left-hand navigation pane:

     

    Release Notes Type Description
    Production

    Release notes for monthly releases and weekly updates that have been deployed to Production for the past three months.

    For older release notes, refer to Archive.

    Preview

    Release notes for monthly releases and weekly updates that have been deployed to Preview.

    For features from previous months that remain in Preview, refer to Additional Preview-only features.
    Early Access

    Release notes for EA opt-in features made available by request to Okta Support.

    Okta Mobile Release Notes Release notes for Okta Mobile and Okta Verify.
    API Release Notes

    API release notes published to the Okta Developer website.
