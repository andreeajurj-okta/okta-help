現在のデバイスまたは新しいデバイスでOkta Verifyをリセットする

現在のデバイスまたは新しいデバイスでOkta Verifyをセットアップするには、まず元の登録を削除する必要があります。 Okta Verifyを削除するには、ブラウザーでOktaアカウントにアクセスする必要があります。 いくつかの制限が適用されます。

注意：このトピックは現在のOkta Verifyユーザーが対象となりますす。 以前デバイスにOkta Verifyをセットアップしたことがない新しいユーザーは、「Okta Verifyをインストールする」をご覧ください。

はじめに

  • 次のいずれかの場合、Okta Verifyをリセットしてください。
    • Okta Verifyをすでにセットアップした後に、現在のデバイスを消去またはリセットした場合。
    • 新しいデバイスに現在のOkta Verifyアカウントをセットアップする場合。
    • 認証ループから抜け出せない場合。ブラウザーでOkta Verifyアプリを確認するよう指示されているが、アプリでプッシュ通知が受信されていないか、またはワンタイムコードが表示されません。
    • アプリから正しいパスコードを入力しても、またはプッシュ通知がデバイスに送信されることを承認しても、アカウントにサインインできない場合。
    • Okta Verifyアプリで認証コードエラーを受信し続けている場合。
  • 既存のOktaアカウントに今でもアクセスでき、次の条件を少なくとも1つ満たしていることをご確認ください。
    • Okta Verifyで登録したデバイスを今でも使用しており、アプリがまだ動作していること。
    • Okta Verifyに加えて、SMS（テキスト）、電子メール 、または（特に）WebAuthNのような認証要素を少な くとも1つ以前にセットアップしている場合。 他の要素をセットアップしたかどうかが分からない場 合に、Oktaにサインインする際、Oktaサインインペー ジのドロップダウンメニューをクリックします。
  • Oktaアカウントにアクセスできない場合、または上記の条件を満たしていない場合、IT部門にお問い合わせください。 一度に1つのデバイスに限りOkta Verifyでセットアップできます。

注意：このトピックについてさらにご質問がありますか？ 「Okta Veifyに関するよくある質問（FAQ）」をご覧ください。

このタスクを開始する

  1. ブラウザーからOktaアカウントにサインインします。 既存のOkta Verifyアプリまたは別の認証要素でサインインできない場合、管理者にお問い合わせください。
  2. [Settings (設定)]を開きます。

    古いダッシュボードを使用している場合、右上のメニューから [Settings (設定)]にアクセスします。

    • [Edit Profile (プロファイルを編集する)]ボタンが表示されたらクリックします。
    • メッセージが表示されたら、パスワードを入力します。
  3. [Extra Verification (追加認証)]セクションに下へスクロールします。

    重要：管理者によって組織用に追加認証要件（多要素認証）が構成されている場合のみ、[Extra Verification (追加認証)]セクションが[Settings (設定)]に表示されます。 詳細については、管理者にお問い合わせください。

  4. Okta Verify横の[Remove (削除)]をクリックします。 これによってデバイスから既存のOkta Verify登録が削除されます。これは、現在のデバイスまたは新しいデバイスでOkta Verifyをリセットする前に必ず必要です。

  5. 削除を確認するメッセージが表示されたら、[Yes (はい)]をクリックします。

  6. [Set up (セットアップする)]をクリックして、登録をリセットするか、新しいデバイスを登録します。

  7. [Set up multifactor authentication (多要素認証をセットアップする)]ページで[Setup (セットアップする)]をクリックします。

  8. [Setup Okta Verify (Okta Verifyをセットアップする)]ページでデバイスの種類を選択して、[Next (次へ)]をクリックします。

  9. デバイスでApp Store（AppleまたはGoogle Play）からOkta Verifyアプリをダウンロードして、インストールします。
  10. デバイスでOkta Verifyを開きます。
  11. アカウントを追加します。
    • iOS
      1. [Add Account (アカウントを追加する)]をタップします。
      2. メッセージが表示されたら、通知を許可し、カメラにアクセスします。
      3. コンピューターのブラウザーで表示されたQRコードをカメラに向けます。 QRコードをスキャンできない場合、「QRコードをスキャンせずにOkta Verifyをセットアップする」をご覧ください。
    • Android
      1. プラス記号（+）をタップします。
      2. コンピューターのブラウザーで表示されたQRコードをカメラに向けます。 QRコードをスキャンできない場合、「 QRコードをスキャンせずにOkta Verifyをセットアップする」をご覧ください。

    QRコードにカメラを向けると、QRコードが自動的に検出され読み取られて、デバイスがOkta Verifyでセットアップされます。

  12. 次の画面がブラウザーで表示されたら、[Finish (完了)]をクリックします。 (この画面は一部の場合にのみ表示されます。)

これでOkta Verifyがデバイスにセットアップされました。Oktaで管理されている組織のアカウントにサインインする際の認証に、これを使用できます。
